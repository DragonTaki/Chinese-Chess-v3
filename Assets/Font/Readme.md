# MOE Li Shu Font (Version 3.00)

## Overview
The **MOE Li Shu Font** is a traditional Chinese typeface based on the "Li Shu" (Clerical Script), developed by the **Ministry of Education, Taiwan (R.O.C.)**. This font file is derived from the *MOE Li Shu Master Script* and is compatible with both Windows and macOS systems. It includes a total of **4,808 Chinese characters**.
